Product engineering and open-source security tools.

### Horizon Security Platform

Embedded intelligence for API security and application defense. All detection and blocking decisions happen locally at the edge. Zero external dependencies.

**Synapse** — Edge detection sensor built on Rust/Pingora. Campaign correlation, bot detection (500+ signatures), DLP scanning, behavioral analysis. Single binary, <1ms detection latency.

**Horizon** — Fleet intelligence and SOC platform. Live threat map, campaign visualization, Sigma rules, war room, sensor management. Self-hosted or SaaS.
